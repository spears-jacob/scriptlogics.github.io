Technical Writing and Documentation Services

Tech stacks are constantly changing in today's business world and documentation is key to success. Whether it is training materials for a growing labor force, presentation materials for clear communication, or maintaining a Docs as Code infrastructure, technical documentation is an often overlooked aspect of Information/Technology workflows. At ScriptLogics, LLC, we provide:

  • Technical Writing - Internal Documentation
    • Process Documentation and Wikis.
    • Project Management documentation.
    • API Documentation.
    • Repository organization and explanation.
    • Application summaries and descriptions.
    • Reporting Structure overviews, explanations, and dictionaries.

  • Technical Writing - External Documentation
    • One-Sheet Application explanations and walkthroughs.
    • User Manuals and Software installation guides.
    • Corporate documentation and Standard Operating Procedures.
    • Business Plans.
    • Requests for Proposals.
    • FAQs.

  • Docs as Code
    • Comprehensive training on Docs as Code methodology, including proper use of Markdown, best practices, and terminology.
    • Converting existing documentation to a Docs as Code infrastructure, including preparing trainging materials on how to utilize and interact with the new documentation infrastructure.
    • Updating and/or consolidating existing Docs as Code infrastructure for readability, organization, and proper Markdown methodology.

  • Training Materials
    • End-to-end Onboarding manuals.
    • Common and complex procedure workflows.
    • Non-technical end-user guides and manuals.
    • Repository-level pipeline synopses.

  • Presentation Materials
    • Slide Decks with summaries.
    • Written communications for non-technical executives and stakeholders.
    • Proof of Concept Dashboards and reporting materials.
